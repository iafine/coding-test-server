## Description

A Blog NodeJS Server used by Coding Test.

- NestJS
- TypeScript
- TypeORM
- Sqlite3
- Yarn

## Installation

```bash
# copy env variable
$ cp .env.example .env

# install dependencies
$ yarn i
```

## Running the app

```bash
# development
$ yarn dev

# production mode
$ yarn prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
